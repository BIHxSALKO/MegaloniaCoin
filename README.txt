README

Lab3 assignment was completed using Python 3.6. Thus, to compile and run, please install Python 3.6 or later.

We create 90 transactions to be stored in a memory pool on the blockchain. Blocks are mined on the blockchain from these transactions. Each block is comprised of 10 transactions: 1 coinbase + 9 txns.

To execute:
$ python3 kchain_megalonia_coin_blockchain.py
